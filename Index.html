Plp Web Dev February Cohort

About the Program

Welcome to an exciting 8-week journey where you'll transform from a coding novice to a confident front-end web developer!

Program Objectives

Understand the core concepts and structure of HTML for building well-structured web pages.
Learn CSS to style and layout web pages, making them visually appealing and user-friendly.
Gain proficiency in JavaScript to add interactivity and dynamic features to your web applications.
Develop problem-solving skills through hands-on projects and real-world scenarios.
Collaborate with peers to enhance teamwork and communication skills in a tech environment.

Course Schedule

Week 1: Introduction to HTML and CSS
Week 2: Advanced CSS Techniques
Week 3: JavaScript Basics
Week 4: DOM Manipulation and Events
Week 5: Responsive Web Design
Week 6: Project Work and Collaboration
Week 7: Final Project Development
Week 8: Presentation and Feedback

Extra Program Information

The program will be self-paced but you will be required to practice daily.
